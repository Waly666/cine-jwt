# cine-jwt

Proyecto sencillo de ejemplo para gestionar autenticación con JWT en un servidor Node.js.

## Descripción

Este proyecto contiene un servidor básico (`server.js`) que sirve como punto de partida
para implementar autenticación usando JSON Web Tokens (JWT). Está pensado para fines
didácticos en el módulo 2.

## Requisitos

- Node.js (versión 14+ recomendada)
- npm

## Instalación

Abre PowerShell en la carpeta del proyecto y ejecuta:

```powershell
npm install
```

## Ejecución

Para iniciar el servidor en modo desarrollo:

```powershell
node server.js
```

Si agregas un script en `package.json`, también puedes usar `npm start`.

## Estructura

- `server.js`: servidor principal.
- `package.json`: dependencias y scripts.
- `README.md`: este archivo.
